# Cutlass - Production Roadmap

## Must-Have for Producing Real Videos

- [ ] **Export to MP4/WebM** - Render timeline to downloadable file via ffmpeg.wasm
- [ ] **Audio playback sync** - Multi-track audio mixing during playback (music bed, VO, SFX)
- [ ] **Real transcription** - Settings panel for Deepgram API key, persist in localStorage
- [ ] **Transitions** - Cross-dissolve and fade-to-black between clips

## High-Value for Ad/Social Production

- [ ] **Text overlays / titles** - Text track with positioning, font, color, timing
- [ ] **Auto-captions** - Burn-in subtitles from transcript, style presets for TikTok/Reels/Shorts
- [ ] **Aspect ratio presets** - 16:9, 9:16, 1:1 with one-click switching
- [ ] **Snap-to-edit-point** - Timeline snapping for aligning cuts to beats/scene boundaries

## Nice-to-Have Polish (Later)

- [ ] Audio waveform rendering (wavesurfer.js)
- [ ] Drag-and-drop between tracks
- [ ] Source monitor with In/Out points
- [ ] Template system for recurring ad formats
